🚀 DEPLOY FRONTEND TO NETLIFY (5 minutes):

STEP 1: Go to Netlify
────────────────────────────────────────────────────────────────────
1. Open: https://app.netlify.com/start
2. Click "Add new site" → "Import an existing project"
3. Choose "Deploy with GitHub"
4. Authorize Netlify if needed

STEP 2: Select Repository
────────────────────────────────────────────────────────────────────
1. Search for: clinisync
2. Click on: rachid-el-adnani/clinisync

STEP 3: Configure Build Settings
────────────────────────────────────────────────────────────────────
Netlify will AUTO-DETECT everything from netlify.toml! ✅

You should see:
   Base directory:      frontend
   Build command:       npm run build
   Publish directory:   frontend/dist
   
Just verify these are correct, then click "Deploy site"

STEP 4: Add Environment Variable
────────────────────────────────────────────────────────────────────
1. Wait for initial deployment to complete
2. Go to: Site settings → Environment variables
3. Click "Add a variable" → "Add a single variable"
4. Add:
   Key:   VITE_API_URL
   Value: https://clinisync-production-e3de.up.railway.app/api
5. Click "Save"

STEP 5: Redeploy Site
────────────────────────────────────────────────────────────────────
1. Go to: Deploys tab
2. Click "Trigger deploy" → "Clear cache and deploy site"
3. Wait ~2 minutes for deployment

STEP 6: Update Railway CORS
────────────────────────────────────────────────────────────────────
1. Copy your Netlify URL (e.g., https://clinisync.netlify.app)
2. Go to Railway → Your clinisync service → Variables
3. Update FRONTEND_URL to your Netlify URL
4. Railway will auto-redeploy
